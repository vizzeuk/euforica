# Videos Folder

Coloca aquí el video `hero-background.mp4` para el Hero Section.

## Especificaciones Recomendadas:

- **Nombre del archivo:** `hero-background.mp4`
- **Formato:** MP4 (codec H.264)
- **Resolución:** 1920x1080 (Full HD) o superior
- **Duración:** 10-30 segundos
- **Peso optimizado:** Menor a 5MB
- **FPS:** 30 o 60
- **Audio:** No necesario (se reproduce muted)
- **Loop:** El video debe verse bien en bucle

## Contenido Sugerido:

- Detalles de eventos elegantes
- Decoraciones sofisticadas
- Iluminación ambiente premium
- Momentos emotivos (sin rostros reconocibles)
- Movimientos de cámara suaves (no abruptos)

## Alternativas si no tienes video:

El componente Hero Section tiene un fallback con gradiente negro elegante.
